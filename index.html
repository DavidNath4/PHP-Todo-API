PHP Todo List API

Check out the full project on GitHub.

Description

This PHP Todo List API is a simple and organized backend service for managing todo items. It provides a set of RESTful endpoints to perform CRUD operations on todo items, making it easy to integrate with various frontend applications.

Concept

The key concepts and features of this API are:

  • Modular Structure
  • Error Handling
  • Database Interaction
  • Validation
  • Interfaces
  • Routing

API Specs

Create Todo

Endpoint: /app.php/createTodo

{
    "title": "Meeting with Client",
    "todo": "Discuss project updates and client requirements",
    "schedule": "2023-09-18"

Response

{
    "message": "Todo item created successfully"
}
